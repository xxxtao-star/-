# 简历匹配系统

一个基于BERT和XGBoost的智能简历匹配系统。

## 🚀 快速开始

### 方式一：快速测试（推荐先运行）

1. 打开文件夹：`简历匹配系统/算法接口/`
2. 双击运行：`快速测试.bat`
3. 按照提示操作

### 方式二：正常启动

1. **启动后端**：
   - 打开：`简历匹配系统/算法接口/`
   - 双击：`启动后端.bat`
   - 等待服务启动成功

2. **启动前端**：
   - 打开：`简历匹配系统/算法接口/`
   - 双击：`启动前端.bat`
   - 浏览器会自动打开

## 📁 文件说明

| 文件/文件夹 | 说明 |
|-----------|------|
| `新建文件夹/app.py` | Streamlit前端应用 |
| `简历匹配系统/算法接口/app.py` | Flask后端服务 |
| `简历匹配系统/算法接口/启动后端.bat` | 一键启动后端 |
| `简历匹配系统/算法接口/启动前端.bat` | 一键启动前端 |
| `简历匹配系统/算法接口/快速测试.bat` | 快速测试工具 |
| `简历匹配系统/BERT模型/` | BERT模型文件 |
| `简历匹配系统/训练数据/` | 岗位和简历数据集 |

## 🔧 依赖安装

如果遇到模块缺失，请运行：

```bash
pip install flask flask-cors transformers torch scikit-learn pandas numpy python-dotenv streamlit requests
```

## 📖 详细说明

详见：`简历匹配系统/使用说明.md`

## ⚠️ 注意事项

- 确保先启动后端，再启动前端
- 后端使用端口5000
- 前端使用端口8501
- 如BERT模型加载失败，系统会自动使用简化匹配

## 🎯 功能特性

- 智能简历解析
- BERT语义匹配
- 关键词匹配
- 岗位推荐
- 简历优化建议
